# Movie-database-management-system
前端参考了https://github.com/lin-xin/vue-manage-system ，感谢大佬的无私开源！  
前端使用Vue框架，后端使用Springboot框架，实现了电影的多样化查询、增加、删除和评分功能  
电影数据来源：imdb.com
